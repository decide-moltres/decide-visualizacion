More information about the plataform

The objective of this project is to implement a secure electronic voting platform,which complies with a series of basic guarantees, such as anonymity and voting secrecy. It is an educational project, designed to study voting systems, so simplicity takes precedence over efficiency whenever possible. Therefore, some shortcomings are ssumed to allow it to be understood and extended.
